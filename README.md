# Inventory Management System in C

A file-based Inventory Management System built using C programming language.

## Features
- Add Products
- Update Stock
- Delete Products
- Search Products
- Persistent Storage using Files

## Tech Used
- C Programming
- Structures
- File Handling
- Arrays
- Functions

## How to Run

gcc main.c product.c file.c -o inventory

./inventory

- File handling in C
- Struct-based design
- Data persistence
- Modular programming
